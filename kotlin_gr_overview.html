Ιστορικά στοιχεία

Τον Ιούλιο του 2011, η εταιρεία JetBrains έφερε στο φως το Σχέδιο Kotlin, μία νέα γλώσσα για τη JVM, που αναπτυσόταν επί ένα χρόνο. Ο επικεφαλής της JetBrains, Dmitry Jemerov, δήλωσε πως οι περισσότερες γλώσσες προγραμματισμού δεν είχαν τα στοιχεία εκείνα που αναζητούσαν, με εξαίρεση τη γλώσσα Scala. Παρόλα αυτά, σχολίασε τον αργό χρόνο μεταγλώττισης της Scala ως μειονέκτημα. Ένας από τους κύριους σκοπούς της Kotlin είναι να μεταγλωττίζει γρήγορα σαν τη Java. Το Φεβρουάριο του 2012, η JetBrains έκανε ανοιχτό τον κώδικα του Project υπό την άδεια Apache 2.

Η ονομασία της γλώσσας προέρχεται από το νησί Kotlin, που βρίσκεται κοντά στο St. Petersburg.Ο Andrey Breslav σημείωσε πως η ομάδα που τη δημιούργησε αποφάσισε να την ονομάσει έτσι λόγω του νησιού, όπως η Java πήρε το όνομά της από την Ιάβα (Java), το νησί της Ινδονησίας (μολονότι η Java πιθανότατα ονομάστηκε έτσι από την ποικιλία καφέ Java παρά εξαιτίας του νησιού).

Η JetBrains ήλπισε πως η νέα αυτή γλώσσα θα δώσει ώθηση στις πωλήσεις του IntelliJ IDEA.

Τι είναι η Kotlin;

Η Kotlin είναι μία μοντέρνα γλώσσα προγραμματισμού, η οποία κυκλοφόρησε το 2016 από την JetBrains.

Έκτοτε έγινε πολύ δημοφιλής, καθώς είναι συμβατή με τη γλώσσα Java (μία από τις δημοφιλέστερες γλώσσες προγραμματισμού που χρησιμοποιούνται), κάτι που σημαίνει πως ο κώδικας της Java (όπως και οι βιβλιοθήκες της) μπορούν να χρησιμοποιηθούν στα προγράμματα που γράφονται με Kotlin.

Η Kotlin χρησιμοποιείται για:

  • Εφαρμογές κινητών (ειδικότερα εφαρμογές για Android)
  • Ανάπτυξη ιστού
  • Εφαρμογές από την πλευρά του εξυπηρετητή (Server side applications)
  • Επιστήμη δεδομένων
  • Και πολλά, πολλά άλλα!

Γιατί να χρησιμοποιήσω Kotlin?

  • Είναι πλήρως συμβατή με τη Java
  • Δουλεύει σε διαφορετικές πλατφόρμες (Windows, Mac, Linux, Raspberry Pi, κλπ.)
  • Είναι περιεκτική και ασφαλής
  • Είναι εύκολη στην εκμάθηση, ιδιαίτερα για όσους έχουν εξοικείωση με τη Java
  • Είναι ελεύθερη προς χρήση
  • Έχει σημαντική τεχνική υποστήριξη

Εκδόσεις της Kotlin

Η πρώτη έκδοση της Kotlin (1.0) κυκλοφόρησε στις 15 Φεβρουαρίου 2016, θεωρήθηκε ως η πρώτη επίσημη σταθερή έκδοση και η JetBrains δεσμεύτηκε για μακροχρόνια συμβατότητα αναδρομικά ξεκινώντας με την συγκεκριμένη έκδοση.

Το 2017 η Google ανακοίνωσε υποστήριξη πρώτης τάξης της Kotlin για τα Android.

Η Kotlin 1.2 κυκλοφόρησε στις 28 Νοεμβρίου του 2017. Διαμοιραζόμενος κώδικας μεταξύ των πλατφορμών JVM και JavaScript μόλις είχε προστεθεί σε αυτήν την έκδοση (ενώ από την έκδοση 1.4 η δυνατότητα προγραμματισμού σε διάφορες πλατφόρμες απετέλεσε κύριο στοιχείο της).

Η Kotlin 1.3 κυκλοφόρησε στις 29 Οκτωβρίου του 2018, φέρνοντας coroutines για ασύγχρονο προγραμματισμό.

Το Μάιο του 2019 η Google ανακοίνωσε πως η Kotlin στο εξής θα είναι η προτιμώμενη γλώσσα των προγραμματιστών εφαρμογών Android.

Η Kotlin 1.4 κυκλοφόρησε τον Αύγουστο του 2020, με μερικές μικρές αλλαγές αναφορικά με την υποστήριξη της πλατφόρμας της Apple, δηλαδή για το διαλειτουργικό Objective-C/Swift .

Η Kotlin 1.5 κυκλοφόρησε το Μάιο του 2021.

Η kotlin 1.6 κυκλοφόρησε το Νοέμβριο του 2021.

Η Kotlin 1.7 κυκλοφόρησε τον Ιούνιο του 2022, περιλαμβάνοντας την A-έκδοση του νέου K2 μεταγλωττιστή της Kotlin.

Η kotlin 1.8 κυκλοφόρησε το Δεκέμβριο του 2022.
