Kotlin is a modern, trending programming language. It is easy to learn, especially if you already know Java (it is 100% compatible with Java). Kotlin is used to develop Android apps, server side apps, and much more.

In this Tutorial sections you can find lots of info which will help you understand Kotlin. There is also Kotlin Tutorial in Greek language (just click on the Greek flag above) and of course you can always visit the official Kotlin website .
