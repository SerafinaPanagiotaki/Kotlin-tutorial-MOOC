Συναρτήσεις κλάσεων

Μπορούμε επίσης να χρησιμοποιούμε συναρτήσεις (functions) μέσα σε μία κλάση, για να εκτελέσουμε συγκεκριμένες ενέργειες: 

// Δημιουργία της συνάρτησης drive() μέσα στην κλάση Car class και κλήση της:
class Car(var brand: String, var model: String, var year: Int) {
    // Συνάρτηση κλάσης
    fun drive() {
        println("Wrooom!")
  }
}

fun main() {
    val c1 = Car("Ford", "Mustang", 1969)
  
    // Κλήση της συνάρτησης
    c1.drive()
}

Υπόδειξη: Όταν μία συνάρτηση δηλώνεται μέσα σε κλάση, είναι γνωστή ως συνάρτηση κλάσης (class function), ή συνάρτηση μέλος (member function).

Παρατήρηση: Όταν δημιουργείται ένα αντικείμενο κλάσης, έχει πρόσβαση σε όλες τις συναρτήσεις της κλάσης.

Παράμετροι συναρτήσεων κλάσεων

Όπως ακριβώς με τις κανονικές συναρτήσεις, μπορούμε να περνούμε παραμέτρους σε μία συνάρτηση κλάσης: 

// Δημιουργία δύο συναρτήσεων: drive() και speed(), και πέρασμα παραμέτρων στην συνάρτηση speed():
class Car(var brand: String, var model: String, var year: Int) {
    // Class function
    fun drive() {
        println("Wrooom!")
    }
                  
    // Class function with parameters
    fun speed(maxSpeed: Int) {
    println("Max speed is: " + maxSpeed)
    }
}
                
fun main() {
    val c1 = Car("Ford", "Mustang", 1969)
    val c1 = Car("Ford", "Mustang", 1969)
  
    // Κλήση των συναρτήσεων
    c1.drive()
    c1.speed(200)
}
